##### 题目描述

有 $N$ 头牛站成一行，被编队为 $1$ 到 $N$，每头牛的身高都为整数。

当且仅当两头牛中间的牛身高都比它们矮时，两头牛方可看到对方。给出其中最高的牛的编号 $P$ 和身高 $H$，剩余牛的身高未知。

给出这群牛之中存在的 $M$ 对关系，每对关系都指明了某两头牛 $X$ 和 $Y$ 可以相互看见。

求每头牛的身高的最大可能值是多少。

##### 样例

###### 样例输入

```java
9 3 5 5
1 3
5 3
4 3
3 7
9 8
```

###### 样例输出

```java
5

5

4
```

##### 考试时思路

根据题意模拟，在区间内除非0元素全部减到适合区间，没有考虑去重。

##### 正确思路

###### 贪心

考虑区间问题，考虑贪心算法排序解决，时间复杂度 $O(n\log{n})$。

初始时整个数组的所有元素均为 $H$

两对对望关系只有两种情况：完全不包含或完全包含。

排序方式：

>  首先考虑按左端点排序，如果相等则情况为两者相互包含。
>
> 若两者相互包含，则先算最外层再算内层才是符合题意情况的，不然将会出错。
>
> 因此左端点相等应按照右端点从大到小排序。

排完序后，若区间内有点大于等于两端点中较小值，则减为两端点的较小值减一，否则不变。

sort的cmp部分代码

```c++
bool cmp(pair<int, int> x, pair<int, int> y)
{
    return (x.first < y.first) || (x.first == y.first && x.second > y.second);
}
```

处理过程代码

```c++
for (int i = 0; i < m; i++)
{
	int Min = min(h1[a[i].first], h1[a[i].second]);
	for (int j = a[i].first + 1; j < a[i].second; j++)
		if (h1[j] >= Min) h1[j] = Min - 1;
}
```
